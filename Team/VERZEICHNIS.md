# Teamverzeichnis — Website

## Ueberblick

Dieses Team arbeitet **innerhalb der Medidyne-Website**. Die Schnittstelle zur
Steuerungsebene ist ausschliesslich der Web-Leiter.

## Projekt

| Aspekt | Detail |
|--------|--------|
| Framework | Next.js 16 (App Router) |
| Sprache | TypeScript |
| Styling | Tailwind CSS v4 |
| Deployment | Vercel (Hobby Account) |
| Domain | medidyne-systems.de |
| Marke | Medidyne Systems GmbH / EmMa.AI |

## Kern-Team

| Rolle | Funktion | Datei | Agent |
|-------|----------|-------|-------|
| **Web-Leiter** | Koordination, einzige Schnittstelle zur Steuerungsebene | `web-leiter.md` | `@web-leiter` |
| **Web-Architekt** | Next.js-Architektur, Komponenten-Strategie, Performance | `web-architekt.md` | `@web-architekt` |
| **Web-Rollen** | Identifiziert Bedarf an neuen Spezialisten, erstellt deren Definitionen | `web-rollen.md` | `@web-rollen` |
